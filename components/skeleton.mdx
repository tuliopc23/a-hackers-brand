# skeleton

Cyberpunk-styled skeleton component with neon effects and terminal aesthetics.

## Import

```svelte
import { skeleton } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<skeleton variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
