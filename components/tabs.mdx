# Tabs

Elegant tabbed navigation with cyberpunk aesthetics, smooth transitions, and neon accent effects.

## Import

```svelte
import { Tabs } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<script>
  import { Tabs } from 'bubble-tea-party'
  
  let tabs = [
    { id: 'system', label: 'System', content: 'System diagnostics and status...' },
    { id: 'network', label: 'Network', content: 'Network configuration and monitoring...' },
    { id: 'security', label: 'Security', content: 'Security protocols and access control...' }
  ]
</script>

<Tabs {tabs} variant="cyberpunk" />
```

## Variants

- **Default** - Clean tab interface
- **Cyberpunk** - Neon accents and glow
- **Terminal** - Terminal-style tabs
- **Neon** - Full neon treatment

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `tabs` | `Tab[]` | `[]` | Tab configuration |
| `variant` | `'default' \| 'cyberpunk' \| 'terminal' \| 'neon'` | `'default'` | Tab style |
| `activeTab` | `string` | `''` | Active tab ID |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Tab layout |
