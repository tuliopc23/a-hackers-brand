# theme selector

Cyberpunk-styled theme selector component with neon effects and terminal aesthetics.

## Import

```svelte
import { themeselector } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<themeselector variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
