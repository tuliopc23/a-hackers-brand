# color palette

Cyberpunk-styled color palette component with neon effects and terminal aesthetics.

## Import

```svelte
import { colorpalette } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<colorpalette variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
