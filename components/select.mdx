# Select

Dropdown selection component with cyberpunk styling, search functionality, and support for single or multiple selections.

## Import

```svelte
import { Select } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<script>
  import { Select } from 'bubble-tea-party'
  
  let options = [
    { value: 'matrix', label: 'The Matrix' },
    { value: 'cyberpunk', label: 'Cyberpunk 2077' },
    { value: 'neuromancer', label: 'Neuromancer' },
    { value: 'ghost', label: 'Ghost in the Shell' }
  ]
  
  let selected = ''
</script>

<Select 
  {options}
  bind:value={selected}
  label="Choose your reality"
  variant="cyberpunk"
/>
```

## Multi-Select

```svelte
<script>
  let selectedItems = []
</script>

<Select 
  {options}
  bind:value={selectedItems}
  multiple={true}
  label="Select multiple items"
  variant="cyberpunk"
  placeholder="Choose your favorites..."
/>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `options` | `Option[]` | `[]` | Available options |
| `value` | `any` | `undefined` | Selected value(s) |
| `multiple` | `boolean` | `false` | Enable multi-select |
| `searchable` | `boolean` | `false` | Enable search functionality |
| `variant` | `'default' \| 'cyberpunk' \| 'neon' \| 'terminal'` | `'default'` | Style variant |
| `label` | `string` | `''` | Select label |
| `placeholder` | `string` | `''` | Placeholder text |
| `disabled` | `boolean` | `false` | Disable selection |
