# Progress

Cyberpunk-styled progress indicators that show system status, loading states, and data transfer progress with authentic terminal aesthetics.

## Import

```svelte
import { Progress } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<script>
  import { Progress } from 'bubble-tea-party'
  
  let progress = 67
</script>

<Progress value={progress} variant="cyberpunk" />
```

## Variants

<Tabs>

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `value` | `number` | `0` | Progress value (0-100) |
| `variant` | `'default' \| 'cyberpunk' \| 'neon' \| 'terminal'` | `'default'` | Style variant |
| `color` | `string` | `'green'` | Neon color for effects |
| `showLabel` | `boolean` | `true` | Show percentage label |
| `animated` | `boolean` | `true` | Enable animations |
