# liquid terminal

Cyberpunk-styled liquid terminal component with neon effects and terminal aesthetics.

## Import

```svelte
import { liquidterminal } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<liquidterminal variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
