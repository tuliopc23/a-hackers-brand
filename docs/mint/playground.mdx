## Code Examples

### Complete Terminal Interface

```svelte
<script>
  import { 
    TerminalWindow, 
    AdvancedTerminal, 
    MatrixRain,
    Button,
    StatusBar 
  } from '@org/ui'
  
  let commands = []
  
  function handleCommand(cmd) {
    commands = [...commands, {
      command: cmd,
      output: `Executing: ${cmd}`,
      timestamp: new Date()
    }]
  }
</script>

<div class="cyber-interface">
  <MatrixRain />
  
  <StatusBar status="online" />
  
  <TerminalWindow title="SYSTEM CONTROL">
    <AdvancedTerminal 
      prompt="neo@matrix:~$"
      {commands}
      onCommand={handleCommand}
      aiEnabled={true}
    />
  </TerminalWindow>
  
  <div class="actions">
    <Button variant="neon">Execute</Button>
    <Button variant="cyberpunk">Deploy</Button>
  </div>
</div>
```

```svelte
  </Card>

```

## Component Combinations

### Form + Validation

```svelte
<script>
  import { 
    Input, 
    Button, 
    FormErrorBoundary,
    Toast 
  } from '@org/ui'
</script>

<FormErrorBoundary>
  <form class="cyber-form">
    <Input 
      label="Access Code"
      type="password"
      variant="cyberpunk"
      required
    />
    
    <Input 
      label="Neural Link ID"
      variant="terminal"
      required
    />
    
    <Button variant="neon" type="submit">
      Jack In
    </Button>
  </form>
</FormErrorBoundary>
```

### Data Display

```svelte
<script>
  import { 
    DataTable, 
    Badge, 
    Progress 
  } from '@org/ui'
  
  let data = [
    { id: 1, status: 'active', progress: 87 },
    { id: 2, status: 'offline', progress: 23 },
    { id: 3, status: 'error', progress: 0 }
  ]
</script>

<DataTable 
  {data}
  columns={[
    { key: 'id', title: 'ID' },
    { 
      key: 'status', 
      title: 'Status',
      render: (value) => `<Badge variant="${value}">${value}</Badge>`
    },
    {
      key: 'progress',
      title: 'Progress',
      render: (value) => `<Progress value="${value}" variant="neon" />`
    }
  ]}
  variant="cyberpunk"
/>
```

## Advanced Patterns

### Modal + Terminal

```svelte
<script>
  import { 
    Modal, 
    AdvancedTerminal,
    Button 
  } from '@org/ui'
  
  let showTerminal = false
</script>

<Button onClick={() => showTerminal = true}>
  Open Terminal
</Button>

<Modal bind:open={showTerminal} variant="terminal" size="lg">
  <AdvancedTerminal 
    prompt="root@matrix:~$"
    aiEnabled={true}
    theme="matrix"
  />
</Modal>
```

## Tips & Tricks

<Tip>
Combine `MatrixRain` with glass variant components for an authentic cyberpunk layered effect.
</Tip>

<Warning>
Don't overuse neon effects - balance bright components with darker elements for better visual hierarchy.
</Warning>

<Info>
All playground examples are fully responsive and work great on mobile devices.
</Info>
