# Components API Reference

Comprehensive API documentation for all Bubble Tea Party components. Each component includes props, events, slots, and usage examples.

## Core Components

### Button

Interactive cyberpunk-styled buttons with multiple variants and effects.

<Fields>
  <Field name="variant" type="string" default="default">
    Button style variant: `default`, `cyberpunk`, `neon`, `ghost`, `glitch`
  </Field>
  <Field name="size" type="string" default="md">
    Button size: `sm`, `md`, `lg`
  </Field>
  <Field name="color" type="string" default="green">
    Neon glow color for neon variant
  </Field>
  <Field name="disabled" type="boolean" default="false">
    Disable button interaction
  </Field>
  <Field name="loading" type="boolean" default="false">
    Show loading state with spinner
  </Field>
  <Field name="href" type="string" default="undefined">
    If provided, renders as anchor link
  </Field>
</Fields>

**Events:**
- `click` - Fired when button is clicked
- `focus` - Fired when button receives focus
- `blur` - Fired when button loses focus

## Terminal Components

### AdvancedTerminal

Full-featured terminal emulator with command history and AI suggestions.

<Fields>
  <Field name="prompt" type="string" default="$">
    Terminal prompt text
  </Field>
  <Field name="commands" type="Command[]" default="[]">
    Array of command history objects
  </Field>
  <Field name="suggestions" type="string[]" default="[]">
    Available command suggestions
  </Field>
  <Field name="aiEnabled" type="boolean" default="false">
    Enable AI-powered command suggestions
  </Field>
  <Field name="theme" type="string" default="matrix">
    Terminal color theme: `matrix`, `amber`, `green`
  </Field>
</Fields>

**Events:**
- `command` - Fired when command is executed
- `suggestion` - Fired when suggestion is selected

## Type Definitions

### Command

```typescript
interface Command {
  command: string
  output?: string
  timestamp?: Date
  exitCode?: number
}
```

### Column

```typescript
interface Column {
  key: string
  title: string
  sortable?: boolean
  render?: (value: any, row: any) => string
  width?: string
}
```

### Theme

```typescript
interface Theme {
  name: string
  colors: {
    primary: string
    secondary: string
    background: string
    surface: string
    text: string
    textSecondary: string
  }
  effects: {
    glowIntensity: 'low' | 'medium' | 'high'
    animationSpeed: 'slow' | 'normal' | 'fast'
    particleEffects: boolean
    scanLines: boolean
  }
}
```
