## Basic Setup

### 1. Import the CSS Theme

Add the cyberpunk theme to your main CSS file or app entry point:

```css
@import 'bubble-tea-party/styles';
```

### 2. Configure Your Theme

Set up the theme provider in your main layout:

```svelte
<!-- +layout.svelte -->
<script>
  import { ThemeProvider } from 'bubble-tea-party'
  import { themeStore } from 'bubble-tea-party/stores'
</script>

<ThemeProvider theme={$themeStore}>
  <main class="cyberpunk-app">
    <slot />
  </main>
</ThemeProvider>
```

### 3. Use Your First Component

Create a simple cyberpunk interface:

```svelte
<!-- +page.svelte -->
<script>
  import { 
    Button, 
    TerminalWindow, 
    GlowText,
    MatrixRain 
  } from 'bubble-tea-party'
  
  let message = 'System Ready'
  
  function initializeSystem() {
    message = 'Welcome to the Matrix...'
  }
</script>

<div class="app-container">
  <MatrixRain />
  
  <TerminalWindow title="SYSTEM CONTROL">
    <GlowText color="neon-green" size="lg">
      {message}
    </GlowText>
    
    <Button 
      variant="cyberpunk" 
      onClick={initializeSystem}
      class="mt-4"
    >
      Initialize
    </Button>
  </TerminalWindow>
</div>
```

## Common Patterns

### Terminal Interface

```svelte
<script>
  import { 
    TerminalWindow, 
    CommandBlock, 
    AdvancedTerminal 
  } from 'bubble-tea-party'
</script>

<TerminalWindow title="root@matrix:~$">
  <CommandBlock command="ls -la">
    drwxr-xr-x  system  root    matrix-files/
    -rw-r--r--  system  root    the-one.exe
  </CommandBlock>
  
  <AdvancedTerminal 
    prompt="matrix@neo:~$"
    suggestions={['help', 'exit', 'wake-up']}
  />
</TerminalWindow>
```

### Data Display

```svelte
<script>
  import { 
    DataTable, 
    MetricsCard, 
    StatusBar 
  } from 'bubble-tea-party'
</script>

<div class="dashboard">
  <StatusBar status="online" />
  
  <MetricsCard 
    title="System Performance"
    value="98.7%"
    trend="up"
  />
  
  <DataTable 
    data={systemLogs}
    columns={['timestamp', 'level', 'message']}
    theme="cyberpunk"
  />
</div>
```

### Interactive Forms

```svelte
<script>
  import { 
    Input, 
    Select, 
    Button, 
    FormErrorBoundary 
  } from 'bubble-tea-party'
</script>

<FormErrorBoundary>
  <form class="cyber-form">
    <Input 
      label="Access Code"
      type="password"
      variant="neon"
      placeholder="Enter the matrix..."
    />
    
    <Select 
      label="Security Level"
      options={['Low', 'Medium', 'High', 'Maximum']}
      variant="cyberpunk"
    />
    
    <Button type="submit" variant="neon">
      Access Granted
    </Button>
  </form>
</FormErrorBoundary>
```

## Tips for Success

<Tip>
Start with the `TerminalWindow` component as your main container - it provides the cyberpunk aesthetic foundation for other components.
</Tip>

<Warning>
Some WebGL components require additional setup. Check the individual component docs for specific requirements.
</Warning>

<Info>
All components support dark mode by default. Use the `ThemeSelector` component to let users toggle themes.
</Info>
