# Troubleshooting

## Tailwind v4 (no PostCSS)
- Ensure `@import "tailwindcss";` and no `postcss.config.*` present.

## LightningCSS
- Set `css.transformer: 'lightningcss'` in Vite.

## ESM externals
- Library build externalizes `react`, `react-dom`, and `svelte`.

## TS strict quirks
- `tsconfig.json` has `strict: true`; fix all implicit anys.

## Storybook/Vite dedupe
- If duplicates occur, align Vite aliases between app and Storybook.
