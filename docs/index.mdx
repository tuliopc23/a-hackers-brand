## Quick Start

<Steps>
  <Step title="Install the Package">
    ```bash
    # Bun (recommended)
    bun add a-hacker-brand
    
    # npm/pnpm
    npm install a-hacker-brand
    pnpm add a-hacker-brand
    ```
  </Step>
  
  <Step title="Install Peer Dependencies">
    ```bash
    bun add svelte @threlte/core @threlte/extras three lucide-svelte
    ```
  </Step>
  
  <Step title="Import and Use">
    ```svelte
    <script>
      import { Button, Card, GlowText } from 'a-hacker-brand';
    </script>
    
    <Card variant="glass">
      <GlowText color="pink" intensity="intense">
        Welcome to A Hacker's Brand
      </GlowText>
      <Button variant="terminal">Get Started</Button>
    </Card>
    ```
  </Step>
</Steps>

## CSS Custom Properties

Customize themes with CSS variables:

```css
:root {
  /* Terminal Theme */
  --terminal-green: #00ff41;
  --terminal-cyan: #00ffff;
  --terminal-blue: #0080ff;
  
  /* Glass Theme */
  --glass-pink: #ff6b9d;
  --glass-purple: #c44af2;
  --glass-blue: #4fa8ff;
  --glass-radius: 16px;
  
  /* Liquid Theme (Experimental) */
  --liquid-glow-intensity: 0.8;
  --liquid-gradient-stops: 3;
}
```
