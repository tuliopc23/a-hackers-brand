# Histoire (Component Previews)

Histoire is set up for this Svelte component library to preview components in isolation.

## Commands

```bash
# Install deps (Bun recommended)
bun install

# Start Histoire
bun run histoire

# Build static preview
bun run histoire:build
```

## File Conventions

- Stories live next to components and use the `*.story.svelte` suffix.
- Example: `src/lib/components/Button.story.svelte`.

```svelte
<script>
  import { Story, Variant } from '@histoire/plugin-svelte'
  import Button from '$lib/components/Button.svelte'
</script>

<Story title="Button">
  <Variant title="Default">
    <Button>Click me</Button>
  </Variant>
  <Variant title="Primary">
    <Button variant="primary">Primary</Button>
  </Variant>
  <Variant title="Sizes">
    <div style="display: grid; gap: 8px; grid-auto-flow: column; align-items: center;">
      <Button size="sm">Small</Button>
      <Button size="md">Medium</Button>
      <Button size="lg">Large</Button>
    </div>
  </Variant>
  <!-- Add more variants as needed -->
  
</Story>
```

## Configuration

Histoire is configured via `histoire.config.ts` with Svelte support:

```ts
import { defineConfig } from 'histoire'
import { HstSvelte } from '@histoire/plugin-svelte'

export default defineConfig({
  plugins: [HstSvelte()],
})
```

You can customize story globs, theme, or Vite options here.

## Notes

- Storybook has been fully removed from scripts and devDependencies.
- Use Histoire for local previews and to generate a static preview for sharing.
