## Usage Patterns

### Basic Component Usage

```svelte
<script>
  import { Button, Card, GlowText } from 'a-hacker-brand';
</script>

<Card variant="glass">
  <GlowText color="pink" intensity="intense">
    Welcome to A Hacker's Brand
  </GlowText>
  <Button variant="terminal">Get Started</Button>
</Card>
```

### Variant Switching

```svelte
<script>
  import { Button } from 'a-hacker-brand';
  
  let currentVariant = 'terminal';
  const variants = ['terminal', 'glass', 'liquid'];
  
  function cycleVariant() {
    const currentIndex = variants.indexOf(currentVariant);
    currentVariant = variants[(currentIndex + 1) % variants.length];
  }
</script>

<Button variant={currentVariant} on:click={cycleVariant}>
  Current: {currentVariant}
</Button>
```

### Responsive Design

```svelte
<script>
  import { Card, Button } from 'a-hacker-brand';
</script>

<div class="responsive-grid">
  {#each items as item}
    <Card variant="glass">
      <h3>{item.title}</h3>
      <p>{item.description}</p>
      <Button variant="terminal">Learn More</Button>
    </Card>
  {/each}
</div>

<style>
  .responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 1rem;
  }
</style>
```
