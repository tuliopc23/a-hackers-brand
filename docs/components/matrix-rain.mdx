# Matrix Rain

The iconic digital rain effect from the Matrix. Creates an animated background of falling green characters that perfectly captures the cyberpunk aesthetic.

## Import

```svelte
import { MatrixRain } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<script>
  import { MatrixRain } from 'bubble-tea-party'
</script>

<div class="container">
  <MatrixRain />
  <div class="content">
    <!-- Your content here -->
  </div>
</div>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `speed` | `number` | `30` | Animation speed (higher = faster) |
| `density` | `number` | `0.6` | Character density (0-1) |
| `color` | `string` | `'#00ff41'` | Rain color |
| `characters` | `string` | `'01'` | Characters to use |
| `fadeEnabled` | `boolean` | `true` | Enable fade effect |
