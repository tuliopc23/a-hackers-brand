# Badge

Compact status indicators and labels with cyberpunk styling. Perfect for showing status, counts, and categories.

## Import

```svelte
import { Badge } from '@org/ui'
```

## Basic Usage

```svelte
<Badge>Default</Badge>
<Badge variant="success">Online</Badge>
<Badge variant="warning">Pending</Badge>
<Badge variant="danger">Offline</Badge>
<Badge variant="neon" color="cyan">Cyber</Badge>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'default' \| 'success' \| 'warning' \| 'danger' \| 'neon'` | `'default'` | Badge style |
| `color` | `string` | `'green'` | Neon color for neon variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Badge size |
