# status bar

Cyberpunk-styled status bar component with neon effects and terminal aesthetics.

## Import

```svelte
import { statusbar } from '@org/ui'
```

## Basic Usage

```svelte
<statusbar variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
