# Card

Sleek cyberpunk containers that organize content with style. Features neon borders, glass morphism effects, and animated backgrounds.

## Import

```svelte
import { Card } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { Card } from '@org/ui'
</script>

<Card>
  <h3>System Status</h3>
  <p>All systems operational. Matrix connection stable.</p>
</Card>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'default' \| 'cyberpunk' \| 'glass' \| 'neon'` | `'default'` | Card style variant |
| `color` | `string` | `'green'` | Neon glow color |
| `padding` | `string` | `'md'` | Internal padding |
| `hoverable` | `boolean` | `false` | Enable hover effects |
| `clickable` | `boolean` | `false` | Make card clickable |
