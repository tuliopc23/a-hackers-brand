# pagination

Cyberpunk-styled pagination component with neon effects and terminal aesthetics.

## Import

```svelte
import { pagination } from '@org/ui'
```

## Basic Usage

```svelte
<pagination variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
