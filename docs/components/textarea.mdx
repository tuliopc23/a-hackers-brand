# textarea

Cyberpunk-styled textarea component with neon effects and terminal aesthetics.

## Import

```svelte
import { textarea } from 'bubble-tea-party'
```

## Basic Usage

```svelte
<textarea variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
