# Accordion

Collapsible content sections that reveal information with style. Perfect for organizing large amounts of data in a cyberpunk interface.

## Import

```svelte
import { Accordion } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { Accordion } from '@org/ui'
  
  let items = [
    {
      title: 'System Diagnostics',
      content: 'All systems operational. Matrix connection stable at 99.7% uptime.'
    },
    {
      title: 'Security Protocols',
      content: 'Firewall active. Intrusion detection enabled. No threats detected.'
    },
    {
      title: 'Data Streams',
      content: 'Processing 2.7TB of data per second. Neural networks fully synchronized.'
    }
  ]
</script>

<Accordion {items} variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `items` | `AccordionItem[]` | `[]` | Array of accordion items |
| `variant` | `'default' \| 'cyberpunk' \| 'neon'` | `'default'` | Styling variant |
| `multiple` | `boolean` | `false` | Allow multiple items open |
| `defaultOpen` | `number \| number[]` | `undefined` | Initially open items |
