# slider

Cyberpunk-styled slider component with neon effects and terminal aesthetics.

## Import

```svelte
import { slider } from '@org/ui'
```

## Basic Usage

```svelte
<slider variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
