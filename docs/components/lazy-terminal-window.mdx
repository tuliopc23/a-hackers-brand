# lazy terminal window

Cyberpunk-styled lazy terminal window component with neon effects and terminal aesthetics.

## Import

```svelte
import { lazyterminalwindow } from '@org/ui'
```

## Basic Usage

```svelte
<lazyterminalwindow variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
