# terminal window

Cyberpunk-styled terminal window component with neon effects and terminal aesthetics.

## Import

```svelte
import { terminalwindow } from '@org/ui'
```

## Basic Usage

```svelte
<terminalwindow variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
