# Modal

Immersive overlay dialogs that command attention in true cyberpunk style. Features glass morphism, animated entrances, and terminal-inspired layouts.

## Import

```svelte
import { Modal } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { Modal, Button } from '@org/ui'
  
  let showModal = false
</script>

<Button onClick={() => showModal = true}>
  Open Modal
</Button>

<Modal bind:open={showModal} title="System Alert">
  <p>Unauthorized access detected. Initiate countermeasures?</p>
  
  <svelte:fragment slot="actions">
    <Button variant="ghost" onClick={() => showModal = false}>
      Cancel
    </Button>
    <Button variant="neon" onClick={() => showModal = false}>
      Execute
    </Button>
  </svelte:fragment>
</Modal>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `open` | `boolean` | `false` | Modal visibility state |
| `title` | `string` | `''` | Modal header title |
| `variant` | `'default' \| 'cyberpunk' \| 'terminal' \| 'glass'` | `'default'` | Modal style |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Modal size |
| `closable` | `boolean` | `true` | Show close button |
| `backdrop` | `boolean` | `true` | Enable backdrop click to close |
