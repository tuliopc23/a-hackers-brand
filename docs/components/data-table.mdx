# Data Table

Advanced data table component with cyberpunk aesthetics, built-in sorting, filtering, and pagination. Perfect for displaying complex datasets in terminal-style interfaces.

## Import

```svelte
import { DataTable } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { DataTable } from '@org/ui'
  
  let data = [
    { id: 1, name: 'Neo', status: 'online', level: 99 },
    { id: 2, name: 'Trinity', status: 'active', level: 87 },
    { id: 3, name: 'Morpheus', status: 'offline', level: 95 }
  ]
  
  let columns = [
    { key: 'id', title: 'ID', sortable: true },
    { key: 'name', title: 'Agent', sortable: true },
    { key: 'status', title: 'Status', render: renderStatus },
    { key: 'level', title: 'Level', sortable: true }
  ]
  
  function renderStatus(value) {
    const colors = {
      online: 'green',
      active: 'cyan', 
      offline: 'red'
    }
    return `<Badge color="${colors[value]}">${value}</Badge>`
  }
</script>

<DataTable 
  {data}
  {columns}
  variant="cyberpunk"
  sortable={true}
  filterable={true}
/>
```

## Features

- **Sorting** - Click column headers to sort
- **Filtering** - Global search and column filters
- **Pagination** - Handle large datasets
- **Row Selection** - Single or multi-row selection
- **Custom Rendering** - Custom cell content
- **Virtual Scrolling** - Performance for large datasets

```svelte
        <Button size="sm" variant="ghost" onclick="restart('${row.id}')">

        </Button>
      `

  ]

  variant="cyberpunk"
  pageSize={10}

  onRowSelect={handleRowSelect}

```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `data` | `any[]` | `[]` | Table data array |
| `columns` | `Column[]` | `[]` | Column definitions |
| `variant` | `'default' \| 'cyberpunk' \| 'neon' \| 'terminal'` | `'default'` | Table style |
| `sortable` | `boolean` | `true` | Enable sorting |
| `filterable` | `boolean` | `true` | Enable filtering |
| `selectable` | `boolean` | `false` | Enable row selection |
| `pageSize` | `number` | `25` | Rows per page |
| `showPagination` | `boolean` | `true` | Show pagination controls |
| `virtualScroll` | `boolean` | `false` | Enable virtual scrolling |

## Column Configuration

```typescript
interface Column {
  key: string                    // Data property key
  title: string                  // Column header text
  sortable?: boolean            // Enable sorting
  width?: string                // Column width
  align?: 'left' | 'center' | 'right'  // Text alignment
  render?: (value, row) => string      // Custom renderer
  filter?: 'text' | 'select' | 'range' // Filter type
}
```

## Events

| Event | Description |
|-------|-------------|
| `rowClick` | Fired when row is clicked |
| `rowSelect` | Fired when row selection changes |
| `sort` | Fired when sorting changes |
| `filter` | Fired when filters are applied |
| `pageChange` | Fired when page changes |
