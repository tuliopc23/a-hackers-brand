# carousel

Cyberpunk-styled carousel component with neon effects and terminal aesthetics.

## Import

```svelte
import { carousel } from '@org/ui'
```

## Basic Usage

```svelte
<carousel variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
