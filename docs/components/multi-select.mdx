# multi select

Cyberpunk-styled multi select component with neon effects and terminal aesthetics.

## Import

```svelte
import { multiselect } from '@org/ui'
```

## Basic Usage

```svelte
<multiselect variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
