# checkbox

Cyberpunk-styled checkbox component with neon effects and terminal aesthetics.

## Import

```svelte
import { checkbox } from '@org/ui'
```

## Basic Usage

```svelte
<checkbox variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
