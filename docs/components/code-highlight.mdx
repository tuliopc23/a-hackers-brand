# code highlight

Cyberpunk-styled code highlight component with neon effects and terminal aesthetics.

## Import

```svelte
import { codehighlight } from '@org/ui'
```

## Basic Usage

```svelte
<codehighlight variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
