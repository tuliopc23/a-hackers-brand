# dropdown

Cyberpunk-styled dropdown component with neon effects and terminal aesthetics.

## Import

```svelte
import { dropdown } from '@org/ui'
```

## Basic Usage

```svelte
<dropdown variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
