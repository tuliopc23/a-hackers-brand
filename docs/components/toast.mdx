# Toast

Temporary notification messages that appear and disappear with smooth cyberpunk animations. Perfect for user feedback and system notifications.

## Import

```svelte
import { Toast, toast } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { toast } from '@org/ui'
  
  function showSuccess() {
    toast.success('Access granted! Welcome to the Matrix.')
  }
  
  function showError() {
    toast.error('Access denied. Invalid credentials.')
  }
  
  function showWarning() {
    toast.warning('System temperature critical.')
  }
</script>

<Button onClick={showSuccess}>Show Success</Button>
<Button onClick={showError}>Show Error</Button>
<Button onClick={showWarning}>Show Warning</Button>
```

## Toast API

```javascript
import { toast } from '@org/ui'

// Different types
toast.success('Success message')
toast.error('Error message') 
toast.warning('Warning message')
toast.info('Info message')

// Custom styling
toast('Custom message', {
  variant: 'cyberpunk',
  duration: 5000,
  position: 'top-right'
})
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'success' \| 'error' \| 'warning' \| 'info' \| 'cyberpunk'` | `'info'` | Toast type |
| `duration` | `number` | `4000` | Auto-dismiss time (ms) |
| `position` | `string` | `'top-right'` | Toast position |
| `dismissible` | `boolean` | `true` | Show close button |
