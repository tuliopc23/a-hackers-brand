# drawer

Cyberpunk-styled drawer component with neon effects and terminal aesthetics.

## Import

```svelte
import { drawer } from '@org/ui'
```

## Basic Usage

```svelte
<drawer variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
