# AI Command Suggestions

Intelligent command suggestion system that provides contextual recommendations and auto-completion for terminal interfaces using AI.

## Import

```svelte
import { AICommandSuggestions } from '@org/ui'
```

## Basic Usage

```svelte
<script>
  import { AICommandSuggestions } from '@org/ui'
  
  let currentCommand = ''
  let context = {
    workingDirectory: '/home/neo',
    previousCommands: ['ls', 'cd matrix', 'whoami'],
    systemInfo: { os: 'linux', shell: 'bash' }
  }
  
  function handleSuggestionSelect(suggestion) {
    currentCommand = suggestion.command
  }
</script>

<AICommandSuggestions 
  bind:input={currentCommand}
  {context}
  onSelect={handleSuggestionSelect}
  variant="cyberpunk"
  aiProvider="openai"
/>
```

## Features

- **Contextual Suggestions** - AI analyzes current directory and history
- **Smart Completion** - Completes partial commands intelligently
- **Error Correction** - Suggests fixes for common command errors
- **Learning System** - Adapts to user patterns over time
- **Multi-Shell Support** - Works with bash, zsh, fish, and PowerShell

## Advanced Configuration

```svelte
<script>
  import { AICommandSuggestions } from '@org/ui'
  
  let aiConfig = {
    provider: 'openai',
    model: 'gpt-4',
    maxSuggestions: 5,
    contextWindow: 100,
    learningEnabled: true,
    customPrompts: {
      systemPrompt: 'You are a helpful terminal assistant...',
      userPrompt: 'Suggest commands for: {input}'
    }
  }
</script>

<AICommandSuggestions 
  {aiConfig}
  showConfidence={true}
  enableExplanations={true}
  variant="matrix"
/>
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `input` | `string` | `''` | Current command input |
| `context` | `CommandContext` | `{}` | Terminal context information |
| `aiConfig` | `AIConfig` | `{}` | AI provider configuration |
| `variant` | `'default' \| 'cyberpunk' \| 'matrix' \| 'terminal'` | `'default'` | Visual style |
| `maxSuggestions` | `number` | `5` | Maximum suggestions to show |
| `showConfidence` | `boolean` | `false` | Show confidence scores |
| `enableExplanations` | `boolean` | `false` | Show command explanations |
| `learningEnabled` | `boolean` | `true` | Enable pattern learning |

## Events

| Event | Description |
|-------|-------------|
| `select` | Fired when suggestion is selected |
| `request` | Fired when AI request is made |
| `response` | Fired when AI response received |
| `error` | Fired when AI request fails |

- **Caching** - Caches frequent suggestions locally
- **Background Processing** - Non-blocking AI requests
- **Fallback Mode** - Works without AI when offline

## Integration Examples

### With AdvancedTerminal

```svelte
<script>
  import { AdvancedTerminal, AICommandSuggestions } from '@org/ui'
  
  let terminalInput = ''
</script>

<AdvancedTerminal>
  <AICommandSuggestions 
    bind:input={terminalInput}
    slot="suggestions"
  />
</AdvancedTerminal>
```

### With Custom Terminal

```svelte
<script>
  import { AICommandSuggestions } from '@org/ui'
  
  let suggestions = []
  
  function handleSuggestions(newSuggestions) {
    suggestions = newSuggestions
  }
</script>

<div class="terminal-interface">
  <input bind:value={command} />
  
  <AICommandSuggestions 
    input={command}
    onSuggestions={handleSuggestions}
  />
  
  {#if suggestions.length > 0}
    <ul class="suggestions-list">
      {#each suggestions as suggestion}
        <li class="suggestion-item">
          {suggestion.command}
          <span class="confidence">{suggestion.confidence}%</span>
        </li>
      {/each}
    </ul>
  {/if}
</div>
```
