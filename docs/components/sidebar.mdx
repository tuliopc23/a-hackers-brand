# sidebar

Cyberpunk-styled sidebar component with neon effects and terminal aesthetics.

## Import

```svelte
import { sidebar } from '@org/ui'
```

## Basic Usage

```svelte
<sidebar variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
