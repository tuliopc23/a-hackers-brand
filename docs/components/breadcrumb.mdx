# breadcrumb

Cyberpunk-styled breadcrumb component with neon effects and terminal aesthetics.

## Import

```svelte
import { breadcrumb } from '@org/ui'
```

## Basic Usage

```svelte
<breadcrumb variant="cyberpunk" />
```
<div class="ahb-preview" data-ahb-component="Breadcrumb" data-ahb-props='{"variant":"cyberpunk"}'>Live: Breadcrumb</div>

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
