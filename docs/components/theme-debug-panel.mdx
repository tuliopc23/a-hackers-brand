# theme debug panel

Cyberpunk-styled theme debug panel component with neon effects and terminal aesthetics.

## Import

```svelte

```

## Basic Usage

```svelte
<themedebugpanel variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
