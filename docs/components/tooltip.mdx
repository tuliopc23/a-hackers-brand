# tooltip

Cyberpunk-styled tooltip component with neon effects and terminal aesthetics.

## Import

```svelte
import { tooltip } from '@org/ui'
```

## Basic Usage

<div data-ahb-component="Tooltip" data-ahb-props='{"variant":"cyberpunk"}'>Live: Tooltip</div>

```svelte
<tooltip variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
