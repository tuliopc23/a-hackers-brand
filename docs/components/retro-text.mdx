# retro text

Cyberpunk-styled retro text component with neon effects and terminal aesthetics.

## Import

```svelte
import { retrotext } from '@org/ui'
```

## Basic Usage

```svelte
<retrotext variant="cyberpunk" />
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Component style variant |
