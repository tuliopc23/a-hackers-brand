# Theming System

Bubble Tea Party's powerful theming system lets you customize every aspect of the cyberpunk aesthetic. From neon colors to glow effects, create the perfect digital atmosphere for your application.

## Theme Provider Setup

```svelte
<!-- +layout.svelte -->
<script>
  import { ThemeProvider } from 'bubble-tea-party'
  import { themeStore } from 'bubble-tea-party/stores'
</script>

<ThemeProvider theme={$themeStore}>
  <slot />
</ThemeProvider>
```

## Theme Switching

```svelte
<script>
  import { ThemeSelector } from 'bubble-tea-party'
  import { themeStore } from 'bubble-tea-party/stores'
  
  const themes = [
    { id: 'cyberpunk-classic', name: 'Classic' },
    { id: 'neon-nights', name: 'Neon Nights' },
    { id: 'matrix-mode', name: 'Matrix' },
    { id: 'cyber-violet', name: 'Violet' }
  ]
</script>

<ThemeSelector {themes} bind:selected={$themeStore} />
```

## Custom Theme Creation

```javascript
// themes/custom-theme.js
export const customTheme = {
  name: 'Custom Cyber',
  colors: {
    primary: '#ff6b35',      // Orange neon
    secondary: '#00d4ff',    // Cyan accent
    background: '#0a0a0a',   // Dark background
    surface: '#1a1a1a',     // Card backgrounds
    text: '#ffffff',         // Primary text
    textSecondary: '#cccccc' // Secondary text
  },
  effects: {
    glowIntensity: 'high',
    animationSpeed: 'fast',
    particleEffects: true,
    scanLines: true
  },
  typography: {
    fontFamily: 'Orbitron, monospace',
    fontSize: {
      sm: '0.875rem',
      md: '1rem', 
      lg: '1.25rem',
      xl: '1.5rem'
    }
  }
}
```

## Theme Variables

### CSS Custom Properties

```css
:root {
  /* Primary Colors */
  --primary-color: #00ff41;
  --secondary-color: #ff00ff;
  --accent-color: #00ffff;
  
  /* Background Colors */
  --bg-primary: #0a0a0a;
  --bg-secondary: #1a1a1a;
  --bg-tertiary: #2a2a2a;
  
  /* Text Colors */
  --text-primary: #ffffff;
  --text-secondary: #cccccc;
  --text-muted: #888888;
  
  /* Effects */
  --glow-size: 10px;
  --glow-intensity: 0.8;
  --animation-speed: 0.3s;
  
  /* Border & Radius */
  --border-radius: 4px;
  --border-width: 1px;
  --neon-border: 2px solid var(--primary-color);
}
```

## Dark Mode Support

```svelte
<script>
  import { darkMode } from 'bubble-tea-party/stores'
  
  // Toggle dark mode
  function toggleDarkMode() {
    $darkMode = !$darkMode
  }
</script>

<button onClick={toggleDarkMode}>
  {$darkMode ? 'Light' : 'Dark'} Mode
</button>
```

## Advanced Customization

### Component-Specific Theming

```svelte
<Button 
  style="
    --btn-bg: linear-gradient(45deg, #ff6b35, #f7931e);
    --btn-glow: 0 0 20px #ff6b35;
    --btn-text: #000000;
  "
>
  Custom Themed Button
</Button>
```

### Runtime Theme Switching

```svelte
<script>
  import { setTheme, getTheme } from 'bubble-tea-party/theme'
  
  function switchToCustomTheme() {
    setTheme({
      colors: {
        primary: '#ff0080',
        secondary: '#8000ff'
      },
      effects: {
        glowIntensity: 'maximum'
      }
    })
  }
</script>
```

## Theme Store API

```javascript
import { themeStore } from 'bubble-tea-party/stores'

// Subscribe to theme changes
themeStore.subscribe(theme => {
  console.log('Theme changed:', theme)
})

// Update theme
themeStore.update(theme => ({
  ...theme,
  colors: {
    ...theme.colors,
    primary: '#new-color'
  }
}))

// Set completely new theme
themeStore.set(newTheme)
```

## Best Practices

<Tip>
Use the theme provider at the root of your application to ensure consistent theming across all components.
</Tip>

<Warning>
Avoid hardcoding colors in component styles. Always use CSS custom properties for theme compatibility.
</Warning>

<Info>
The theme system automatically handles accessibility requirements like contrast ratios and focus indicators.
</Info>
